# Share Wheels Website

Multi-page marketing site for Share Wheels.

## Pages

| Route | Page |
|-------|------|
| `/` | Home |
| `/features` | Features |
| `/how-it-works` | How it works |
| `/drivers` | For drivers |
| `/download` | Download app |

## Assets

- **Logo only (local):** `/public/logo-mark.png`, `/public/favicon.png`
- **All other images:** Unsplash URLs in `src/data/images.js`

## Run

```bash
cd share-wheels-web
npm install
npm run dev
```

Open http://localhost:5174

## Build

```bash
npm run build
npm run preview
```
